# /debug

1. Read `.claude/skills/debug/SKILL.md` and follow it as the canonical workflow.
2. Restate the symptom exactly as observed.
3. Gather logs, traces, and reproduction steps before theorizing.
4. Reproduce with the smallest reliable path.
5. Apply the smallest fix that explains the evidence.
6. Run the narrowest validation that proves the fix.
7. Report root cause, fix, validation evidence, and remaining risk.
